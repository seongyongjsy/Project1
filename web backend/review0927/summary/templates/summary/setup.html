가상환경 설정 및 장고 설치

가상환경 설정

가상환경(Virtual Environment)은 한 컴퓨터 내에서 독립적인 공간을 마련해서 개발환경을 설정하는 것을 말한다.

파이썬의 경우, 수 많은 라이브러리(=모듈)을 지원하는 장점이 있지만, 한 환경에 사용하지 않는 라이브러리를 설치하고 파이썬을 실행하면 모든 라이브러리를 불러오며 성능 상 불이익이 발생하기에 이를 회피하기 위해서, 또한 많은 수의 라이브러리를 관리하기 용이하게 하기 위해서 가상환경을 사용하곤 한다.

파이썬은 기본적으로 venv 모듈을 통해 가상환경을 지원한다.

가상환경 설정 방법

  1. 가상환경을 설치할 경로 지정 및 설치
    • (명령 프롬프트에서)cd C:\
    • (명령 프롬프트에서)mkdir venvs
    • (명령 프롬프트에서)cd venvs
    • (명령 프롬프트에서)py -m venv webapp-env
  2. 가상환경 실행 및 종료
    • (명령 프롬프트에서)cd C:\venvs\webapp-env\Scripts
    • (명령 프롬프트에서)activate
    • (명령 프롬프트에서)deactivate
  3. VS Code에서 가상환경 설정 방법
    • (VS Code에서)ctrl + shift + P 입력 후 select interpreter 검색
    • (VS Code에서)Python: Select Interpreter 선택
    • (VS Code에서)+ 인터프리터 경로 입력... 선택
    • (VS Code에서)찾기... 선택
    • (탐색기 창에서)C:\venvs\webapp-env\Scripts\python.exe 선택
    • (VS Code에서)ctrl + ` 입력 후 터미널 창 확인

장고 설치

장고는 파이썬 모듈로써 설치한다.

2022년 9월 26일 기준 4.1.1 버전이 최신 버전이다.

x.2 버전이 LTS 버전이다. 일반적으로 LTS 버전 사용을 권장한다.

장고 설치 방법

  1. 가상환경을 실행한 후 설치를 진행한다.
    • (명령 프롬프트에서)py -m pip install django
    • (명령 프롬프트에서)py -m pip install django==3.2.15

설치된 장고 버전 확인

    • (명령 프롬프트에서)py -m pip show django
    • (명령 프롬프트에서)py -m django --version
    • (파이썬 스크립트에서)import django; print(django.get_version())
