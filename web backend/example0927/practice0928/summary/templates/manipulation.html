데이터베이스 조작

작성한 모델 클래스를 이용해서 데이터베이스를 조작할 수 있다.

모델 클래스의 objects 객체를 사용해서 SELECT, INSERT 등의 쿼리를 자동으로 수행할 수 있다. 

        from board.models import Board
    
        Board.objects.all()  # SELECT 대응 함수. 모든 데이터를 가져옴.
        Board.objects.get(num=1)  # SELECT ... WHERE 대응 함수. 조건에 맞는 데이터를 가져옴.
        Board.objects.create(...)   # INSERT 대응 함수. 입력할 데이터를 키워드 인수로 작성.
