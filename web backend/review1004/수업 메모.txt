HTTP 상태 응답 코드(HTTP Status Response Code):
- 서버로부터 응답을 받은 결과, 정상적으로 응답을 받았는지, 혹은 오류가 발생했는지 등 각각의 상황에 대해 임의의 숫자를 부여해서 지정하는 방식.

- 주요 코드
  - 200: 정상.
  - 404: Page Not Found, 요청한 URL에 대응되는 패턴을 찾을 수 없을 때.
  - 405: Method Not Allowed, URL 패턴은 GET만 허용하도록 작성했는데 요청은 POST로 했을 때 등.
  - 500: Internal Server Error, 백엔드에서 문법적 오류가 발생했을 때.


관리자 페이지
- 사이트 관리를 위해 장고에서 기본적으로 제공하는 기능.
- 특히 데이터베이스의 데이터를 직접 조작할 수 있다.

관리자 페이지 조작 순서
1. 관리하기 위한 앱을 settings.py의 INSTALLED_APPS에 등록한다.
2. 앱 폴더 내 admin.py에 모델 클래스를 등록한다.
  - admin.site.register(모델클래스)
  - admin.site.register([클래스1, 클래스2, ...])
3. 관리자 계정 생성.
  - py manage.py createsuperuser
  - 관리자 계정명, 이메일 주소(선택), 비밀번호를 입력한다.
4. 관리자 페이지 접속.
  - http://127.0.0.1:8000/admin/

  - 요청-응답 처리(URL 패턴, 뷰 파일 등)
