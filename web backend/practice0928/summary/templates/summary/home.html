데이터베이스 연동 이론 정리 및 예제 프로젝트

장고는 ORM 방식으로 데이터베이스 연동 기능을 지원함.

ORM(Object Relational Mapping)

데이터베이스 연동 전용 객체(=모델)를 사용하는 방식. 모델 클래스를 작성해서 데이터베이스의 테이블을 나타내고, 해당 클래스의 객체를 사용해서 데이터베이스의 데이터를 조작하는 방법.

데이터베이스 접속 설정, 쿼리문 등을 자동으로 설정해주기 때문에 해당 부분을 크게 신경쓰지 않아도 되어서 애플리케이션 개발에만 집중할 수 있는 장점이 있음.
