CSS 박스 모델

  • 어떤 HTML 요소든, 콘텐츠 영역, 패딩 영역, 테두리 영역, 마진 영역을 가질 수 있음.
  • 기본적으로 HTML 요소들은 서로의 영역을 침범하지 않는 형태로 표현됨.
  • 따로 설정하지 않으면 기본적으로 패딩, 테두리, 마진 영역은 없지만 일부 HTML 요소는 자동으로 설정됨.
  • 콘텐츠, 패딩, 테두리, 마진 영역과 관련된 속성들과 이와 관련된 문법들을 통틀어 CSS 박스 모델이라고 부름.
  • 마진(margin): 외부 여백.
  • 테두리(border): 콘텐츠의 경계.
  • 패딩(padding): 내부 여백.
  • 콘텐츠 영역(content area): 실제로 내용물을 표시하는 영역.
