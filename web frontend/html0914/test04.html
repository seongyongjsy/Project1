CSS 적용 우선 순위

  • 중요도(Importance)
    스타일 시트 종류에 따라 우선 순위를 부여해서 최우선 순위의 CSS를 적용하는 방식.
    1. 사용자 환경 스타일 시트: 접근성 관련 설정 등.
    2. !important가 적용된 스타일: 개발자가 작성하고 !important 키워드를 부여한 스타일.
    3. 개발자 작성 스타일 시트: 개발자가 직접 작성한 스타일.
    4. 브라우저 기본 스타일 시트: 브라우저에서 기본적으로 제공하는 스타일.
  • 명시도(Specificity)
    선택자 종류에 따라 점수를 부여해서 가장 높은 점수를 가진 선택자의 CSS를 적용하는 방식.
    1. 인라인 스타일 시트: 1000점
    2. ID 선택자: 100점
    3. 클래스, 속성 선택자: 10점
    4. 요소 선택자: 1
