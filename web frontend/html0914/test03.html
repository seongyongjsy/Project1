선택자(Selector)

CSS를 지정할 대상을 선택할 때 사용하는 문법.

  • 전체 선택자: * 기호를 이용해서 모든 요소 선택.
  • 요소 선택자: 태그명을 이용해서 모든 해당 요소들 선택.
  • ID 선택자: # 기호와 id 속성값을 이용해서 해당 요소 선택.
  • 클래스 선택자: . 기호와 class 속성값을 이용해서 모든 해당 요소들 선택.
  • 자식 선택자: 부모 태그명, > 기호, 자식 태그명을 이용해서 모든 해당 요소들 선택.
  • 후손 선택자: 부모 태그명, (공백) 기호, 자식 태그명을 이용해서 모든 해당 요소들 선택.
  • AND 조건 선택자: 여러 개의 선택자를 붙여서 모든 선택자를 만족하는 모든 해당 요소들 선택.
  • OR 조건 선택자: 여러 개의 선택자 중 하나라도 만족하는 모든 해당 요소들 선택.
  • 속성 선택자: [] 기호를 이용해서 특정 속성을 가진 모든 해당 요소들 선택.
