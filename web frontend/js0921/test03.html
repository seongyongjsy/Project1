제이쿼리(jQuery)

자바스크립트 라이브러리.

라이브러리(Library)

프로그램의 개발을 돕기 위한 도구(변수, 함수, 클래스, ...)들의 모음.

개발자가 기능을 만드는데 필요한 함수 등을 가져와서 사용할 수 있음.

ex) 제이쿼리, API, numpy, pandas, matplotlib, ..., React

프레임워크(Framework)

프로그램의 개발을 돕기 위한 도구(변수, 함수, 클래스, ...)들의 모음

프레임워크에서 제시하는 개발 방향성을 개발자가 따라서 만들어야 함.

ex) 장고(Django), Bootstrap , 스프링(Spring), 전자정부프레임워크, Vue.js
